My Name is Alex

And this page has a little bit more detail about how I've designed and implemented my websites in the past

This website is built from Pure CSS with a little bit of Javascript (mainly for the media slider and embedded video). I chose Pure over Bootstrap because I wanted something simple, clean and responsive. Pure CSS focused on exactly that. CSS. I wanted to add my own peices of javascript if required, rather than have some of it already implemented.

I built the website from scratch, with inspiration from the layouts page that Pure CSS has. This is a massive improvement over the last site I made, which was cobbled together from a w3 Schools layout. I couldn't follow half the code in it, the CSS and formatting was over engineered and worst of all, it felt dated four months after I completed it.

The old website (v1.0) is here

In the future, I plan to redo my website periodically, because I like tinkering with it and it's a good excuse to learn the latest and greatest web tools as they come out. I'll probably keep storing the old ones as well, so I can build up a history of what my websites have looked like at different times in my career.
